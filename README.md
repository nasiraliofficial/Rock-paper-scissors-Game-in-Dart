# 🎮 Rock Paper Scissors Game

A simple Rock Paper Scissors command-line game built with Dart.

## Features

* Play against the computer
* Random computer choices
* Live score tracking
* Input validation
* Quit anytime with `q`

## Rules

* Rock (`r`) beats Scissors (`s`)
* Scissors (`s`) beats Paper (`p`)
* Paper (`p`) beats Rock (`r`)
* Same choices result in a tie

## Getting Started

### Prerequisites

Install the Dart SDK.

### Run the Game

```bash
dart run main.dart
```

## Controls

| Input | Action   |
| ----- | -------- |
| r     | Rock     |
| p     | Paper    |
| s     | Scissors |
| q     | Quit     |

## Example

```text
------Welcome to the Rock Paper Scissors Game------

Current Score:
You wins 0 times and computer wins 0 times

Enter your choice:
r

You chose r
Computer chose s

You win!
```
